
# Sleep Disorder Classification 🧠💤

## 📌 Project Description
This project predicts sleep disorders using machine learning techniques. 
It uses an Artificial Neural Network (ANN) model trained on sleep and lifestyle data.

## 🚀 Features
- Data preprocessing
- ANN model training
- Sleep disorder prediction

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow / Keras

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the program:
   python SleepDisorder.py

## 📂 Dataset
- cleaned_sleep_data.csv

## 📊 Model
- sleep_disorder_ann_model.h5
